Border

Permite definir en una única regla todos los bordes de un elemento seleccionado. además tiene parámetros para poderlo configurar

Border-left, Border-right, Border-top, Border-bottom

Permite definir el borde de un lado del elemento seleccionado.

Border radius

Sirve para redondear las equinas de un elemento.

background image

Se utiliza para establecer un fondo de pantalla con una imagen

Color

Cambia el color del texto

Font family

Selecciona el tipo de fuente de la página, donde habrá por lo general diferente fuentes si por algún motivo la primera fuente no funcione o no se encuentre disponible.

Font style y Font weight

Font style establece un estilo de fuente en un elemento de texto. Puede ser normal, itálica y oblicuas. Mientras que Font weight establece el grosor del texto.

Width y Height

Width establece el ancho de un elemento, mientras que el height es la altura de un elemento

List style

Configura todas las propiedades de estilo de una lista.

  • pan
  • leche
  • huevos

a.- Hyperlinks

Los hipervínculos permiten vincular documentos a otros documentos o recursos entre paginas web y recursos en línea. Atreves de un enlace.

Por ejemplo de esto es deja un enlace de una canción como la siguiente Cuando Pase el Temblor de Soda Stereo

p.- Parrafo

Los párrafos se emplean para la agrupación de palabras, elementos de HTML como imágenes , listas, enlaces, etc. Permitiendo que el formato de los contenidos sea más estructurado.

Ejemplo de ello: “Si vis pacem, para bellum”, “si quieres la paz, prepárate para la guerra”. Frase de Publio Flavio Vegecio

h1,h2,...,h6.- Encabezados

Los encabezados son utilizados en HTML para la definición de títulos y subtítulos en un pagina web. Dónde viene por niveles donde el “h1” es el mas grande de todos y el ultimo de ellos el “h6” es el mas pequeño.

Ejemplo:

div.- Contenedor de bloque

Los div son contendedores, capaces de almacenar cualquier tipo de elementos y donde se puede aplicarles un estilo en conjunto. Donde realmente sirve para dividir o seleccionar el contenido en bloques. Donde sean manipulados por separado.

Ejemplo de codigo:

            
            <div=”contenedor”>
                <h3> Imagen de resident evil 4 remake </h3>
                <p> leon y Ashley siendo atacados por los ganados </p>
                < src= "https://e00-elmundo.uecdn.es/assets/multimedia/imagenes/2023/03/24/16796882612307.jpg"  
                width="700"
                height="550"
                alt= "Resident evil 4 remake">
            </div>

Es un contenedor en línea, donde su uso es en los párrafos para la aplicación de estilos o agrupar elementos en línea, en lugares específicos del texto.

strong.- texto mayor importancia

Dentro de su contenido resalte la gran importancia en término semántico, seriedad o urgencia. Suelen resaltar en negritas.

Ejemplo es el siguiente:

El juego no tiene el sistema de autoguardado, así que no se le olvide guarda su partida.

i.- Cursiva

Cuando se usa indica que su contenido debe mostrarse en cursiva, mayormente cuando se usa una lengua extranjera.

Ejemplo:

Para escribir poder imprimir alguna cosa en python se utiliza print

u.- Subrayado

Este elemento sirve para que el intervalo del texto seleccionado se muestre subrayado

Ejemplo:

Al morir en la partida,siempre reaparecerás en el ultimo punto de guardado

b.- Negritas

Solamente se utiliza para llamar la atención del contenido al lector, no da importancia semántica, como seria el strong

Ejemplo:

Vive con orgullo y la cabeza alta

strike/del.- Tachador

El elemento strike se utilizaba para tachar texto, sin embargo se ha vuelto obsoleto de usar ya que la mayoría de los navegadores de web ya no son compatibles con esta etiqueta, aun siendo que HTML 5 lo acepte. En cambio se utiliza del siendo semejante la función de tachar texto.

Ejemplo con strike y del:

Estoy usando strike | Yo estoy usando del

small.- Texto pequeño

La etiqueta se utiliza para indicar que el texto dentro de contenido se debe mostrar con una fuente más pequeña a la que se esta utilizando, mayormente utilizando como texto secundario o pies de páginas.

Ejemplo:

pagina de mdn web docs
aviso de privacidad del sitio web

em.- Enfatizar texto

Se utiliza para para enfatizar texto, a diferencia con i, em representa el énfasis del estrés de su contenido.

Ejemplo:

Ahora me he convertido en la muerte, el destructor de mundos

header.- Encabezado

Representa el contenido introductorio, por lo general se incluye titulo de la pagina,navegación principal, la información del autor o cualquier otra información que se considere importante poner al inicio.Siendo el ejemplo seria el inicio de esta pagina web donde se ver implementado el uso del header

Si lo queremos ver el header de nuestra pagina web se veria de esta forma:

                    <time>06.04.2023</time>
                    </header>
                    <p><small>Descripcion de las etiquetas de html</small></p>
                </article>

table.- Tablas

Es un elmento que se utiliza para la creación de una tabla, pero requiere mas elementos como los son

  • <tr> inicia una fila de la tabla
  • <th> inicia una celda de encabezado de la tabla
  • <td> inicia una celda de datos de la tabla

ul.- Lista no ordenada

la creacion de una lista que no posee algun tipo de orden, es decir que no tiene una secuencia en especifico.

Ejemplo:

ol.- Lista ordenada

la creacion de una lista que posee orden, tiene una secuencia en especifico.
